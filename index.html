OAuth Code Receiver

When building integrations to third party API services thru OAuth, it can be useful to have an always on web service to redirect clients to, after authorizing. This can be used for manually collecting the authorization code, and using it server side where you would rather not expose services to the internet. For more info, check out a project that leverages this site: webexAuther serverside OAuth shim for Cisco WebEx..
